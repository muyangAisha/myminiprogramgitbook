# 订单支付

#### **接口基本信息**

* **接口描述**

  订单支付

* **请求方式**

  POST

* **URL**

  /users/pay/{token}

* **接口使用权限校验**

  不需登录

#### **请求参数**

| 名称 | 类型 | 必选 | 说明 |
| --- | --- | --- | --- |
| token | String | Yes | 用户唯一token |
| orderUuid | String | Yes | 订单号 |
| cash | Double | No | 奖金支付金额 |
| wxMoney | Double | No | 微信支付金额 |
